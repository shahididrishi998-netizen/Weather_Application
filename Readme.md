🌦️ AeroGlass Weather

A premium weather dashboard inspired by modern glassmorphism design principles. AeroGlass Weather combines beautiful UI animations with accurate weather forecasting powered by Open-Meteo.

✨ Features

• Real-time weather data
• Dynamic weather-based themes
• Animated SVG weather icons
• Hourly forecast
• 7-day forecast
• Air Quality Index (AQI)
• UV Index
• City search with autocomplete
• Responsive mobile-first design
• Glassmorphism user interface
• Express.js backend
• Open-Meteo weather integration
• No API key required

🛠️ Tech Stack

Frontend:
- HTML5
- CSS3
- Vanilla JavaScript

Backend:
- Node.js
- Express.js

Data Provider:
- Open-Meteo API

Deployment:
- Render
